b tag

This tag is used to display the written text in bold format. It is strictly a presentational elemant. If you want to show your text in bold laetters and not have real semantic meaning, then put it within ... tag.

i tag

The i tag is defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

u tag

This tag represents some text that is unarticulated and styled differently form normal text, such as misspelled words or proper names in Chinese text. The content inside is typically displayed with an underline. You can change this with CSS

select tag

The select element is used to create a drop-down list. The select element is most ogten used in a form, to collect user input. seThe name attribute is needed to referemce the form, to collect user input. The name attribute is needed to reference the form data after the form is submitted (if you omit the name attribute, no data from the drop-down list will bw submitted).

button tag

Inside a button elelment you can put text (and tags like i, b, strong , br, img, etc.). That is not possible with a button ctreated with the input element.

Tip Always specify the type attribute for a button element, to tell browsers what type of button it is.

hr tag

The hr tag in HTML is used to give a thematic break betweeen paragraphs.We use the hr tag to break the flow of content by drawing a horixontal line between paragraphs.

For example, we can use the hr tag for a scene change while writing a play or story.

br tag

The br tag insterts a single line break.
The br tag is useful for writing addresses or poems.
The br tag is an empty tag which means that it has no end tag.
